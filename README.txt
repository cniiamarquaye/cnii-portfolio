# Nii Amarquaye Commey's Portfolio

Welcome!  
This is the personal portfolio of **Nii Amarquaye Commey (PhD)**, showcasing my work in geospatial science, environmental sustainability, and hydrogeology. Here you'll find interactive project summaries, visualizations, and resources for collaboration.

---

## 🚀 About This Site

- Built using the [Massively](https://html5up.net/massively) template by HTML5 UP.
- Designed for clarity, accessibility, and professional presentation.
- Features sections for About, Projects, and Contact.

---

## 📂 Featured Projects

- **Modelling Groundwater Contamination:** Simulating contaminant fate and transport in aquifers using COMSOL.
- **Pesticide Transport and Reaction in Soil:** Coupled flow and chemical reaction modeling for pesticide behavior.
- **Terzaghi Compaction:** Vertical compaction simulation in aquifers due to groundwater withdrawal.
- More projects and research coming soon!
